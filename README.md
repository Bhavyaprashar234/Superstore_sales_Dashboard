# 📊 Superstore Sales Dashboard

## About the Project

This is my first Power BI dashboard project, created to analyze sales performance using the Superstore dataset. The goal of this project was to practice data visualization, DAX, and dashboard design while converting raw sales data into meaningful business insights.

The dashboard allows users to interact with the data using filters and quickly understand sales trends, customer behavior, and regional performance.

---

## Dashboard Overview

The dashboard includes:

* Total Sales, Profit and Quantity KPIs
* Average Delivery Time
* Sales by Category
* Sales by Sub-Category
* Sales by Customer Segment
* Sales by Shipping Mode
* Monthly Sales Trend
* Monthly Profit Trend
* Sales Forecasting
* State-wise Sales Map
* Region-wise filtering using slicers

## Key Insights

* Consumer customers generated the highest sales.
* Standard Class was the most commonly used shipping mode.
* Office Supplies contributed significantly to total revenue.
* Sales showed noticeable growth during the festive months.
* Regional filters help compare sales performance across different parts of the country.
